## [Version 0.1.0-alpha.1](https://github.com/prismrpc/prism/releases/tag/v0.1.0-alpha.1) (2025-12-05)

### major
* initial commit: [`2bdb95d`](https://github.com/prismrpc/prism/commit/2bdb95d)

* API key authentication with rate limiting
* Prometheus metrics and structured logging
* Docker and binary deployment support
